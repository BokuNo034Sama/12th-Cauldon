# 12th Cauldron

Collaborative savings and investment platform for trusted savings circles, treasury transparency, and milestone-based wealth building.

## Workspace

- `apps/web`: Next.js web application. Implemented in Step 02.
- `apps/mobile`: Flutter integration placeholder. Flutter source will live here later.
- `packages/*`: shared runtime packages.
- `tooling/*`: shared TypeScript, ESLint, and Prettier configuration.

## Commands

```bash
pnpm install
pnpm dev
pnpm lint
pnpm typecheck
pnpm build
```
